
  # Marketing Analysis Web System

  This is a code bundle for Marketing Analysis Web System. The original project is available at https://www.figma.com/design/LPDUytdz7fIyLcALaWjhcT/Marketing-Analysis-Web-System.

  ## Production Architecture

  - **Frontend**: `src/` (React + Vite) - Deployed on Vercel
  - **Backend**: `backend/` (FastAPI + Python) - Deployed on Railway
  - **Legacy**: `frontend/` directory is DEPRECATED - Do NOT use

  See `ARCHITECTURE.md` for deployment details and `CLAUDE_RULES.md` for development guidelines.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
